# How to Use Penny at Lataamo

A one-page infographic-style guide for Lataamo team members using Penny in Slack.

## Local preview

```bash
python3 -m http.server 8080
```

Open http://localhost:8080

## GitHub Pages

This repo is configured to publish the root static site via GitHub Pages using the workflow in `.github/workflows/pages.yml`.

After pushing to GitHub:

1. Go to **Settings → Pages**.
2. Set **Build and deployment → Source** to **GitHub Actions**.
3. Run or wait for the **Deploy static site to GitHub Pages** workflow.
